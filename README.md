Twitter clone in SwiftUI, 
Firebase for Login/Register.


- Using Firebase 
  - Login with email
  - Register with email
- Firestore
  - Save all details of register user
  - Upload profile picture
- Tweet like
- Post a tweet
- Fetch Profile detail
- Fetch all tweets
- Fetch likes tweets
- Fetch all register users
- Logout
